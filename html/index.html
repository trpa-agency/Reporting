Tahoe Regional Planning Agency · Cumulative Accounting

Where to Start ?
The TRPA Cumulative Accounting dashboard ecosystem.

Seven dashboards, each focused on one question. Color-coded by audience: Executive for headlines, Operations for per-commodity detail, Trends for year-over-year, Audit for per-APN lineage.

Several dashboards have alternative layouts (Original / Option 1 / Option 2). Each card below lists the available variants. The eyebrow link inside each dashboard lets you switch between them at any time.

Pick the question you want to answer. All numbers are live from the TRPA Cumulative Accounting service - no stale spreadsheets in the loop.

What do you want to know?

Each card is one question with the dashboard that answers it. Color-coded by audience.

Executive
Tahoe Development Tracker
How much has been built? How much more can be built?
Total built + banked + remaining allocations for each commodity. See whether Tahoe is approaching the Regional Plan caps.
Layouts Option A·four sections of cards Option C·three tiles + banked section
Operations
Allocation Tracking
Has growth stayed within the limits of the Regional Plan? What's the balance of each rights pool?
Per-commodity accounting against the Regional Plan caps: total authorized, allocated, jurisdiction pool, TRPA pool, unreleased.
Layouts Option A·card grid by commodity Option B·stacked-bar tiles + hover explainers
Operations
Pool Balance Cards
How is each rights pool tracking against its cap, by jurisdiction?
One card per pool, by jurisdiction. Max, assigned, and what's still available. Residential pools open a per-year metering chart.
Layouts Option A·always-visible explainer panel Option B·explainer on button hover, compact numbers
Trends
Residential Additions by Source
Where did the rights come from for every residential unit built in a year?
5-source breakdown (Allocations / Bonus Units / Transfers / Conversions / From Banked) plus a removed-units chart for the full ledger.
Layouts Option A·facet + combined toggle Option B·facet only + per-source hover explainers
Trends
Development History
How much was added or removed each year?
Year-over-year net change per commodity + the cumulative trajectory. Year slider and jurisdiction pills scope the basin or one county.
Layouts Option A·standard KPIs + charts Option B·compact numbers + KPI hover tooltips
Audit
QA Change Rationale
Why did this number change?
5,925-row audit log of 2023 + 2026 QA correction campaigns. Filterable by APN, change type, year.
Audit
Genealogy Solver
How is this parcel connected to others?
APN lineage lookup. Single APN or batch CSV; renders parcel polygons on a map with 2025 cross-reference inline.
